### 🛡️ My Journey
[**My journey into building this repo started with a setback: I failed my 2nd Dan test in September 2023 because my poomsae quality wasn’t consistent, and I hadn’t fully memorized all fifteen forms. Instead of letting that moment define me, I used it as fuel. I built E‑Sa‑Bum‑Nim, a Python tool that uses text‑to‑speech to call out every movement in a form so I could drill with precision and accountability. Six weeks later, I crushed my 2nd Dan retest—and the same tool became a huge part of how I prepared for and passed my 3rd Dan test months later. Around that time, I got curious about computer vision and realized something important: even strong practitioners (including me) had subtle errors in our blocks and transitions that were hard to see with the naked eye. That curiosity turned into tkd‑drift‑analyzer, a biomechanics project that compares “gold standard” and user videos, extracts joint angles into CSV, and quantifies technical drift. What started as a personal fix for a failure has grown into a set of tools that blend Taekwondo, computer science, and my drive to improve.**]

### 🥋 Featured Projects

#### 📜 e-sa-bum-nim
**https://github.com/ymodak25/e-sa-bum-nim**
A Python project utilizing `pyttsx3` to audio-guide Taekwondo practitioners through complex forms. Designed to facilitate independent practice.

#### 📊 tkd-drift-analyzer
**https://github.com/ymodak25/tkd.drift.analyzer**
A Python-based analysis tool designed to measure the accuracy of Taekwondo movements. By analyzing video footage, this project calculates angles of movements and compares them against a reference "ideal" form to detect inaccuracies. It acts as a digital coach to ensure precision in techniques.
